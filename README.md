              Sistema de Agência Bancária – API REST

API REST desenvolvida com Java e Spring Boot para simular operações bancárias.

                        Funcionalidades

Criar conta bancária

Listar contas

Depositar valor

Sacar valor

Transferir entre contas

Verificar saldo

                       Tecnologias Utilizadas

Java

Spring Boot

Maven

Postman (testes de requisição)

                            Endpoints

GET	/ContaBancaria/status	Verifica status da API
POST	/ContaBancaria	Criar conta
GET	/ContaBancaria	Listar contas
PUT	/ContaBancaria/depositar	Depositar
PUT	/ContaBancaria/sacar	Sacar
                                    Como executar

Clonar o repositório

Abrir na IDE

Rodar a aplicação

Acessar:

http://localhost:3000/ContaBancaria/status
